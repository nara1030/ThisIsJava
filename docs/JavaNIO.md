## 소개
IO와 NIO는 데이터를 입출력한다는 목적은 동일하지만, 방식에 있어서 크게 차이가 난다.

| 구분 | IO | NIO |
| -- | -- | -- |
| 입출력 방식 | 스트림 방식 | 채널 방식 |
| 버퍼 방식 | 논버퍼 | 버퍼 |
| 비동기 방식 | 지원 안함 | 지원 |
| 블로킹 방식 | 블로킹 방식만 지원 | 블로킹/논블로킹 모두 지원 |
