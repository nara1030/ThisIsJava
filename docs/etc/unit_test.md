## 개념
> 단위 테스트란 무엇인가

* 단일[1] 메소드, 함수, 클래스, 독립된[2] 기능 등을 테스트
* 애플리케이션의 한 부분이 의도한대로 동작하는지 검증

> 왜 단위 테스트를 작성하는가


> 설정 및 작성법


- - -
* [1]  
	```txt
	Test only one code unit at a time
	```
* [2]  
	```txt
	Make each test independent to all the others
	```
